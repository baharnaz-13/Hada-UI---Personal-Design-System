Introduction

Welcome to Hada Design System. This guide outlines the design principles, components, and style rules used in our interfaces.

Use this documentation to ensure consistency and alignment across products using the Hada brand.

Color_Palette

The Hada color palette defines the visual identity of the brand. It consists of primary, secondary, and neutral tones.

These colors ensure readability, accessibility, and aesthetic harmony across all UI elements.

  • Primary: #4A90E2 (blue)
  • Accent: #F5A623 (orange)
  • Neutral: #F0F2F5 (light gray)
background-color: #4A90E2; color: #FFFFFF;
Typography

Typography in Hada is designed for clarity and readability. The base typeface is clean, modern, and web-safe.

Font sizes follow a consistent scale to ensure visual hierarchy across text content.

  • Font family: "Inter", sans-serif
  • Base font size: 16px
  • Heading scale: 1.25x multiplier
font-family: 'Inter', sans-serif; font-size: 1rem;
Buttons & Interactions

Buttons are essential UI components that trigger actions. Each button style should follow the Hada guidelines.

Buttons must have clear states: default, hover, active, and disabled.

  • Primary button
  • Secondary button
  • Ghost button
.btn-primary { background: #4A90E2; color: white; } .btn-disabled { opacity: 0.5; cursor: not-allowed; }
Forms & Inputs

Form elements include input fields, checkboxes, radio buttons, and select menus.

Labels must always be associated with form fields to ensure accessibility.

  • Text input
  • Email input
  • Password input
<input type="email" placeholder="you@example.com" /> <label for="name">Name</label>
Spacing & Layout

Hada uses a 4px spacing scale to create rhythm and visual alignment.

Layout is built using a combination of CSS Flexbox and Grid for flexible structure.

  • Spacing unit: 4px
  • Container max width: 1200px
  • Breakpoints: 768px, 1024px
margin: 16px; display: grid;
